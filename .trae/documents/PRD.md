# 点名系统 H5 UI 产品需求文档（PRD）

## 1. 产品概述

本项目是一款面向商场/物业巡检场景的移动端 H5 点名系统，用于对楼层商户的员工到岗情况进行快速签到与统计。核心目标是在手机单屏内完成"看进度 → 切楼层 → 逐户签到 → 看统计"的全流程操作，并通过商户详情页查看本月每日考勤汇总与导出。
- 主要用户：商场/物业现场巡检人员
- 解决问题：传统纸质点名效率低、统计难、易遗漏
- 核心价值：单屏可视化点名 + 月度考勤汇总导出

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 巡检员 | 系统分配账号 | 点名签到、查看进度、查看统计、查看商户详情 |
| 管理员 | 系统分配账号 | 含巡检员全部权限，额外可编辑商户、导出考勤 |

### 2.2 功能模块
1. **点名首页**：标题天气区、今日点名进度卡片、楼层筛选标签栏、商户签到卡片、今日统计底部栏、底部导航栏
2. **商户详情页**：商户头部卡片、功能按钮组、商户基础信息卡片、本月每日考勤卡片

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 点名首页 | 顶部标题天气区 | 展示日期、场地天气、关闭按钮 |
| 点名首页 | 今日点名进度卡片 | 展示 4/11 进度、36% 进度条、环形进度圈、点名率 |
| 点名首页 | 楼层筛选标签栏 | 横向滑动筛选 全部/1F/2F/3F/4F + 筛选按钮 |
| 点名首页 | 商户签到卡片 | 展示楼层标签、商户名、第 X/11 户、签到大圆、底部左右滑操作、分页指示点 |
| 点名首页 | 今日统计底部栏 | 展示今日统计、正常/缺勤/未点名人数、查看详情按钮 |
| 点名首页 | 底部导航栏 | 点名/商户/记录 三点位切换 |
| 商户详情页 | 商户头部卡片 | 绿色头部展示商户名、认证标识 |
| 商户详情页 | 功能按钮组 | 查看点名/编辑商户/联系商户/更多操作 |
| 商户详情页 | 商户基础信息卡片 | 展示商户基础信息 |
| 商户详情页 | 本月每日考勤卡片 | 月份下拉、导出按钮、每日考勤表格、状态标签 |

## 3. 核心流程

用户进入点名首页 → 查看今日进度卡片了解整体点名率 → 通过楼层标签栏切换楼层 → 在商户签到卡片左右滑动逐户签到 → 查看今日统计底部栏 → 点击查看详情进入商户详情页 → 选择月份查看每日考勤表格 → 点击导出本月考勤。

## 4. 用户界面设计

### 4.1 设计风格
- 主色：#00b578（绿色，签到/激活态）
- 警告色：#ff9500（橙色，异常/缺勤）
- 文本黑：#1a1a1a，文本灰：#666666
- 浅灰背景：#f5f5f5，卡片背景：#ffffff
- 卡片圆角：16px，全局左右边距：16px
- 商户名称字号：22px，签到大圆尺寸：170px
- 商户副标题与大圆垂直间距：20px
- 按钮风格：胶囊圆角，绿色底白字
- 布局风格：单屏紧凑卡片式布局，无长滚动
- 图标风格：lucide-react 线性图标，简约现代

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 点名首页 | 顶部标题天气区 | 紧凑日期、场地天气+图标、圆形关闭按钮 |
| 点名首页 | 今日点名进度卡片 | 左侧图标+标题，中间 4/11+进度条，右侧环形圈 |
| 点名首页 | 楼层筛选标签栏 | 胶囊标签，激活绿色，横向滑动，筛选按钮靠右 |
| 点名首页 | 商户签到卡片 | 楼层标签、22px 商户名、170px 渐变签到大圆、左右滑操作、分页点 |
| 点名首页 | 今日统计底部栏 | 左标题+问号、中数据、右查看详情按钮 |
| 点名首页 | 底部导航栏 | 点名/商户/记录 三点位，图标+文字并排 |
| 商户详情页 | 商户头部卡片 | 绿色渐变头部，商户名+认证√ |
| 商户详情页 | 功能按钮组 | 四个功能按钮横向排列 |
| 商户详情页 | 商户基础信息卡片 | 商户基础信息字段 |
| 商户详情页 | 本月每日考勤卡片 | 月份下拉+导出按钮、表格表头、每日数据行、状态标签 |

### 4.3 响应式
- 移动端优先（Mobile First），设计稿基准 375px 宽
- 触控优化：签到卡片支持左右滑动手势，按钮最小触控区 44px
- 单屏展示：点名首页所有模块一屏内完整显示，无超长滚动

### 4.4 3D 场景指引
- 本项目无 3D 场景需求
